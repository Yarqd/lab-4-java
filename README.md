# WhoWantsToBeAMillionaire

Swing-приложение по мотивам игры "Кто хочет стать миллионером?".

## Что реализовано

- 15 уровней вопросов и денежная лестница.
- Вопросы хранятся в SQLite.
- Таблица рекордов и вывод TOP 10 игроков.
- Выбор несгораемой суммы перед стартом игры.
- Подсказки: 50/50, помощь зала, звонок другу, право на ошибку, замена вопроса.
- Динамическая генерация вопроса через Gemini API.

## Запуск

```bash
./gradlew run
```

При первом запуске создаётся база `WhoWantsToBeAMillionaire.db` и заполняется вопросами из `src/main/resources/questions.tsv`.

## AI-генерация вопросов

Для генерации вопросов используется Google Gemini API. Ключ не хранится в проекте, его нужно передать через переменную окружения:

```bash
export GEMINI_API_KEY="ваш_ключ"
./gradlew run
```

В окне игры нажмите кнопку `AI-вопрос`. Приложение сгенерирует вопрос текущего уровня, сохранит его в таблицу `questions` и сразу покажет игроку.

По умолчанию используется модель `gemini-3.5-flash`. При необходимости её можно заменить:

```bash
export GEMINI_MODEL="gemini-3.1-flash-lite"
./gradlew run
```

## Подготовка базы без запуска GUI

```bash
./gradlew run --args="--init-db"
```
